# 05/ - 俄罗斯方块 AI（启发式与 DQN）

这个目录包含俄罗斯方块游戏的 AI 实现，涵盖启发式算法、深度 Q 网络（DQN）、CNN 架构和 ResNet 变体。

## 文件清单

### 核心模块

- **01.py**: 启发式俄罗斯方块 AI
  - 手工设计特征：高度、空洞数、不平整度
  - 权重优化：网格搜索或简单优化
  - 动作评估：枚举所有可能位置

- **02.py**: DQN 俄罗斯方块
  - 深度 Q 网络
  - 经验回放缓冲区
  - ε-贪婪策略
  - 目标网络稳定训练

- **03.py**: CNN 俄罗斯方块
  - 卷积神经网络提取棋盘特征
  - 端到端学习
  - 状态表示：棋盘二维矩阵

- **04.py**: ResNet 俄罗斯方块
  - 残差连接解决梯度消失
  - 更深的网络结构
  - 更好的特征提取能力

- **game.py**: 俄罗斯方块游戏环境
  - 棋盘状态管理
  - 方块生成与旋转
  - 碰撞检测
  - 行消除逻辑
  - 游戏结束判断

- **quixo.py**: Quixo 棋盘游戏
  - 5×5 棋盘
  - 推块机制
  - 胜负判定（连线检测）
  - 双人对战逻辑
