# 09/ - 俄罗斯方块 AI（MLP-Mixer 架构）

这个目录包含俄罗斯方块游戏的 AI 实现，使用 MLP-Mixer 架构进行策略和价值预测，结合蒙特卡洛树搜索（MCTS）。

## 文件清单

### 核心模块

- **agent.py**: AI 智能体
  - 整合 MCTS 和 MLP-Mixer 网络
  - 动作选择逻辑
  - 自对弈生成训练数据
  - 模型更新接口

- **game.py**: 俄罗斯方块游戏环境
  - 棋盘状态管理
  - 方块生成与旋转
  - 碰撞检测
  - 行消除逻辑
  - 游戏结束判断

- **mcts.py**: 蒙特卡洛树搜索
  - 选择：UCB1 公式
  - 扩展：神经网络预测先验概率
  - 模拟：快速 rollout 或使用网络
  - 回溯：更新节点统计信息
  - `c_puct`: 探索常数
  - `n_playout`: 模拟次数

- **model.py**: MLP-Mixer 神经网络
  - Token 混合：跨空间位置信息交换
  - Channel 混合：跨通道信息交换
  - 策略头：输出动作概率分布
  - 价值头：输出局面评估
  - 无卷积的纯 MLP 架构

- **selfplay.py**: 自对弈引擎
  - MCTS 指导的自我对弈
  - 生成 (state, policy, value) 三元组
  - 数据保存到文件
  - 温度参数控制探索

- **train.py**: 训练循环
  - 加载自对弈数据
  - 策略损失 + 价值损失
  - 数据增强
  - 模型检查点保存

- **test.py**: 模型评估
  - 与旧版本对战
  - 胜率统计
  - 性能指标记录

### 数据目录

- **data/**: 自对弈生成的训练数据
- **model/**: 训练好的模型检查点
